# README - Projet Learn@Home

## Description
Ce projet consiste en la conception et le développement d'un site web pour Learn@Home, une association qui met en relation des enfants en difficulté scolaire avec des tuteurs bénévoles. L'objectif est de permettre à tout élève d'avoir accès à un soutien scolaire à distance, en facilitant la communication entre les élèves et les bénévoles.

## Objectifs du projet
1. Concevoir un site web pour Learn@Home qui répond aux besoins de l'association.
2. Faciliter la communication entre les élèves et les bénévoles.
3. Mettre en place une plateforme de soutien scolaire en ligne.
4. Livrer des documents clés pour la validation du projet avec le client.

## Livrables

1. **Diagrammes de cas d'usage** (PDF) : Comprendre les cas d'utilisation pour chaque fonctionnalité majeure du site (connexion, chat, calendrier, gestionnaire de tâches, tableau de bord).

2. **User Stories avec critères d'acceptation** (PDF) : Décrire les fonctionnalités du site sous forme de scénarios utilisateur avec des critères d'acceptation.

3. **Maquettes du site** (PDF) : Présenter des maquettes pour le design du site, incluant au moins une maquette par page. Les maquettes doivent être conçues pour les versions desktop et mobile.

4. **Tableau Kanban** (Notion, Trello ou GitHub) : Organiser le projet en blocs de fonctionnalités et sous-fonctionnalités pour le développement, en utilisant un tableau Kanban. La colonne "ANALYZE" du tableau doit être complétée.
